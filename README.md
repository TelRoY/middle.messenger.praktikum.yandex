# MyMate - Modern Messenger Project
MyMate - это современный, быстрый и безопасный мессенджер с интуитивно понятным интерфейсом. Проект разработан с использованием современных веб-технологий и компонентного подхода.

## Технологии
Vite - быстрый сборщик проектов

Handlebars - шаблонизатор для компонентного подхода

PostCSS - обработка CSS с современными возможностями

Vanilla JavaScript - чистая реализация без фреймворков

CSS Variables - кастомные свойства для единого дизайна

## Начало работы
Предварительные требования
Node.js версии 14 или выше

npm или yarn

## Разработка

### Установка зависимостей
Для установки зависимостей, выполните команду:
```sh
npm install
```

### Запуск Development сервера
Чтобы запустить сервер для разработки, выполните команду:
```sh
npm run dev
```
После запуска откройте браузер и перейдите по адресу:
```sh
http://localhost:3000
```

### Создание билда
Чтобы выполнить production сборку, выполните команду: 
```sh
npm run build
```
Собранные файлы будут находиться в папке dist/.

### Предпросмотр собранного проекта:

```sh
npm run preview
```

### Сборка и запуск проекта:
```sh
npm run start
```

## Доступные страницы

### Основные страницы

1. Главная страница - /

  Описание мессенджера

  Навигация по всем страницам

2. Авторизация - /src/pages/authorization/authorization.html

  Форма входа в аккаунт

  Поля для логина и пароля

  Ссылка на регистрацию

3. Регистрация - /src/pages/registration/registration.html

  Форма создания нового аккаунта

  Поля: имя, фамилия, логин, email, пароль, телефон

4. Главный чат - /src/pages/home/home.html

  Интерфейс мессенджера

  Список чатов (левая колонка)

  Область сообщений (правая колонка)

### Страницы ошибок

5. 404 - Страница не найдена - /src/pages/404/404.html

  Стилизованная страница 404 ошибки

  Кнопка возврата на главную

6. 500 - Внутренняя ошибка сервера - /src/pages/500/500.html

  Стилизованная страница 500 ошибки

  Кнопка возврата на главную
